## 📖 About This Project

This project is a **Huawei Certification Study Guide web application** designed to help learners prepare for:

- HCIA (Associate Level)
- HCIP (Professional Level)
- HCIE (Expert Level)

It covers **openEuler Linux**, **openGauss Database**, and **Huawei Cloud technologies** with structured labs, command references, and interactive quizzes.

## 📚 What You Will Learn

### openEuler
- Linux system administration
- Shell scripting
- Storage management (LVM, partitions)
- Networking & services
- Apache, NGINX, DNS, MySQL

### openGauss
- SQL & database design
- System catalogs & performance tuning
- Security (SSL, RLS, auditing)
- Backup, restore & disaster recovery

### Ceph (HCIE Level)
- Object storage (RGW)
- Block storage (RBD)
- File storage (CephFS)
- Cluster management & CRUSH maps
